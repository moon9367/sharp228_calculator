### 포장비 0원 (박스 포장 가능)
- 짧은변 ≤ 599mm **AND** 긴변 < 600mm
- 또는 착불 발송 조건 미달시

### 예시
- **50×1500**: 긴변 1400~1599, 짧은변 ≤100 → **3,000원**
- **300×2400**: 긴변 2200~2400, 짧은변 200~299 → **12,000원**
- **500×500**: 착불 조건 미달 → **0원**
- **700×700**: 긴변 600~699, 짧은변 600~699 → **3,000원**
